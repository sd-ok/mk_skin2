## Phase 1 — 공통 기반 (CSS Variables & Base)

**목적**: 모든 페이지에서 공유하는 CSS 변수, Reset, 공통 컴포넌트를 정의.

**작업 범위**:

```
skin_dev/common/
  ├── variables.css      ← STYLE_GUIDE 기반 모든 CSS 변수
  ├── reset.css          ← 메이크샵 기본 스타일 충돌 방지 Reset
  ├── base.css           ← body, a, img, button 기본 스타일
  ├── typography.css     ← 타이포그래피 유틸리티
  ├── components.css     ← 공통 컴포넌트 (버튼, 배지, 카드 셸)
  └── utilities.css      ← only-pc / only-mo, sr-only 등 유틸리티
```

**세부 작업**:
- [ ] CSS Custom Properties (`:root`) 전체 정의
- [ ] Tailwind CDN 도입 검토 및 충돌 테스트
  - Tailwind 기본 Preflight(reset)과 메이크샵 충돌 여부 확인
  - 충돌 시 `@layer` 또는 `important: false` 옵션 검토
- [ ] `.only-pc` / `.only-mo` 반응형 유틸리티 (기존 스킨 방식 유지)
- [ ] `sr-only` (스크린리더 전용) 클래스
- [ ] 버튼 스타일 (Primary, Outline, Ghost)
- [ ] 입력 폼 기본 스타일

**수락 기준**:
- [x] CSS 변수가 STYLE_GUIDE.md와 100% 일치
- [x] PC/모바일 분기 유틸리티 정상 동작
- [x] Tailwind: 공통 기반은 순수 CSS 확정. 개별 페이지에서 선택 활용

**완료 파일 (6개)**:
- `skin_dev/common/variables.css` — 전체 CSS Custom Properties (토큰)
- `skin_dev/common/reset.css` — 메이크샵 충돌 방지 Selective Reset
- `skin_dev/common/base.css` — body, #wrap, 컨테이너, sc_script
- `skin_dev/common/typography.css` — 텍스트 스타일 유틸리티
- `skin_dev/common/components.css` — 버튼, 배지, 상품카드 셸, 수량 조절, Swiper 커스텀
- `skin_dev/common/utilities.css` — only-pc/mo, sr-only, flex 헬퍼, 테마, 브레드크럼

## Phase 3 — 메인 페이지

**목적**: 쇼핑몰의 첫인상. 고급스럽고 인터랙티브한 메인 페이지 구현.

```
skin_dev/main/
  ├── main.css
  ├── main.js
  └── main.html
```

### 섹션 구성 (위→아래)

```
1. 히어로 섹션 (Hero)
   ├── 전체화면 비디오 배경 (WebM/MP4)
   ├── 텍스트 오버레이 (쇼핑몰명, 태그라인)
   ├── CTA 버튼 (쇼핑하기)
   └── GSAP ScrollTrigger: 스크롤에 따라 텍스트 슬라이드인

2. 카테고리 쇼케이스
   ├── 돈육 카테고리 카드 (블랙/다크레드 테마)
   ├── 수산 카테고리 카드 (다크블루 테마)
   └── 호버 시 이미지 줌 + 텍스트 페이드인

3. Best 상품 섹션
   ├── 제목 + 화살표 네비게이션
   └── Swiper 가로 슬라이더 (PC: 6개, 모바일: 2.5개)

4. 브랜드 스토리 / 키비주얼 배너
   └── 풀와이드 이미지 + 텍스트 (Scrollytelling 효과)

5. New 상품 섹션
   └── Swiper 슬라이더

6. 추천 상품 섹션
   └── 그리드형 (4열/2열)

7. (선택) 인스타그램 피드 스타일 이미지 그리드
```

**GSAP 활용**:
- 히어로: ScrollTrigger pin + 텍스트 stagger 등장
- 카테고리 쇼케이스: 스크롤 진입 시 좌우 슬라이드인
- 상품 그리드: stagger 0.08s 간격 페이드인

**수락 기준**:
- 비디오 히어로 정상 재생 (autoplay, muted, loop)
- GSAP 애니메이션 부드럽게 동작
- 모든 상품 가상태그 정상 출력
- 모바일에서 비디오 → poster 이미지 폴백 (용량 고려)
